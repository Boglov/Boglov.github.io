~ What is Python? ~

Python is a very easy to learn, powerful and popular programming language. In fact in 2017 it was deemed the most popular programming language in the world by the IEEE Spectrum*, and that still remains true today. We will be using Python 3 which is the newest version.

You can learn to program through the power and ease of Python, and we are here to guide.
