Variables

~ Commenting your code ~

In case you're not familiar with the concept of variables, they are simply containers which can hold your data so you can use it later in your code. To create a variable you just have to come up with a name for it which describes what it will be storing, and set it equal to the value/data you wish to store inside it. For example:

# variable_name = data to store
favorite_candy = "chocolate bars"

The first line in the code above is what's called a comment. A comment is just meant to clarify what your code is doing and while it isn't necessary to comment your code, it is very wise. Commenting your code makes it easier, for you or anyone else who reads your code, to understand what that code is doing. In Python a comment is begun with a pound symbol, #, and for better readability you should follow the pound by one space before typing in your comment as I've done above.

Keep in mind that comments actually have no effect on the functionality of your code and is actually ignored by the Python interpereter when you run it. Only the line with the pound sign will be commented, any following lines of code will be executed as usual. There are a couple different ways to comment your code, including ways to make multi-line comments without using a pound sign at the start of each one, but for now we will stick to the basics.

~ Storing your data in variables ~

In the last tutorial we learned how to store a line of text in a variable we named my_phrase. Text is not the only thing you can store in variables though, variables can hold all sorts of data which can then be used later when it is needed.

Some data types which variables can hold include integers, floats, strings, and booleans. It's very easy to create variables to hold your data, and it's a key part of programming. Below are a few examples:

# Storing an integer (whole number)
my_number = 28
print("My integer:", my_number)

# Storing a float (floating point number)
my_float = 3.75
print("My float:", my_float)

# Storing a boolean (either true or false)
my_bool = True
print("My bool:", my_bool)

# Storing a string (text)
my_string = "Hello, World!"
print("My string:", my_string)

Output:
My integer: 28
My float: 3.75
My bool: True
My string: Hello, World!

You should go ahead and type in the code above if you haven't already, and then run it. The output should be the same as that shown above. That's how easy it is to create variables, but there are a few more types of data you might want to store. Most other data types are just combinations of these basic ones shown above. A simple example of a more complex data type is what's called a tuple. Tuples consist of multiple pieces of data stored together seperated by commas. Like so:

# Storing a tuple of integers (multiple integers seperated by commas)
my_integer_tuple = (3, 5, 8, 2)
print("Integer tuple:", my_integer_tuple)

# Storing a tuple of floats (multiple floats seperated by commas)
my_float_tuple = (1.4, 5.3)
print("Float tuple:", my_float_tuple)

# Storing a tuple of strings (multiple strings seperated by commas)
my_string_tuple = ("dude", "totally", "yeah")
print("String tuple:", my_string_tuple)


Output:
Integer tuple: (3, 5, 8, 2)
Float tuple: (1.4, 5.3)
String tuple: ("dude", "totally", "yeah")
