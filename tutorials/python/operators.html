Using Operators

~ What are Operators? ~

In Python operators are the symbols you use to do math and other operations with variables. It shouldn't be to hard to catch on so I'll just jump into the code and we can go over what it does.

# Store some numbers into variables
n1 = 30
n2 = 5

# Add n1 and n2 and store it's result
result = n1 + n2

# Store the whole equation into a string
# using string formatting
equation_str = "%d + %d = %d" % (n1, n2, result)

# Print the whole equation
print(equation_str)

Output:
30 + 5 = 35
>>

On lines 2 and 3 above we store some numbers into the variables n1 and n2. Then on line 6 we add n1 and n2, with the addition operator +, and store the solution into the result variable.

Next we use what's called String Formatting, on line 10, to make a string which holds the whole equation. This is done by using the %d operator. These operators are essentially placeholders for integers. They are replaced with the contents of the variables we supplied after the string in quotation "%d + %d = %d".

By following that quotation with a % operator and a Tuple containing our numbers (n1, n2, result) we have told the Python interpereter to replace each %d with the variables in the Tuple in the order which they are entered.

Some handy String Formatting operators are:

  • %s - for Strings and other objects (can also be used for numbers)
  • %d - for Integers
  • %f - for Floating point numbers

There are a couple more but for now these will do. For this tutorial we'll only be using the %d operator, as far as String Formatting goes, to put our numbers and results into a string to represent our equations. From here we will go over the remaining math operators as we did for addition.
