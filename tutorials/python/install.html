~ Installing Python 3 through Pyzo ~

To be specific this tutorial series focuses on the version of Python called Python 3. Whichever sub-version is the newest should be the one you install but anything below Python 3 will likely not be entirely compatible with the code in these tutorials.

I will mostly be focused on the Python code itself as it will run the same throughout all editors, or IDEs. Though for beginners I recommend using an IDE named Pyzo as it's compatible with Windows, Linux and MacOS, and has many useful features. I've chosen Pyzo as it will allow me to explain IDE specific steps for beginners which will be the same despite which OS you are using. Although most other IDEs will have very similar steps and shouldn't be too hard to follow along with.

The steps for installing Pyzo are available at the "Getting started with Pyzo" link below. It will guide you through installing the Pyzo IDE, of course, and also through the process of setting it all up with a Python interpereter.
